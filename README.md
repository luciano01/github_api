## For developers
> To clone locally
```
$ git clone git@github.com:luciano01/github_api.git
$ cd github_api
```
> Open the project in an editor, if you use VS Code, follow this
```
$ cd github_api
$ code .
```

> To run, follow this on terminal
```
$ flutter clean
$ flutter pub get
$ flutter run --release
```
